# gitnostr

**Git bridge to Nostr** for [gittr](https://gittr.space). Same codebase as **[arbadacarbaYK/gitnostr](https://gittr.space/arbadacarbaYK/gitnostr?branch=main)** on gittr.

**Components:** `git-nostr-bridge` · `git-nostr-ssh` · `git-nostr-db` · `git-nostr-cli` (`gn`)

**Docs on gittr:** [Architecture](https://gittr.space/arbadacarbaYK/gitnostr?file=docs/ARCHITECTURE.md&branch=main) · [README](https://gittr.space/arbadacarbaYK/gitnostr?branch=main) · [SSH guide](https://gittr.space/arbadacarbaYK/gitnostr?file=SSH_GIT_GUIDE.md&branch=main)

**gittr forge:** [arbadacarbaYK/gittr](https://gittr.space/arbadacarbaYK/gittr?branch=main) · [bridge setup](https://gittr.space/arbadacarbaYK/gittr?file=docs/GIT_NOSTR_BRIDGE_SETUP.md&branch=main)

**Build here (monorepo):**

```bash
make git-nostr-bridge git-nostr-ssh
make git-nostr-cli   # gn
```
