# gitnostr

**Git bridge to Nostr** for [gittr](https://gittr.space). Same codebase as **[arbadacarbaYK/gitnostr](https://github.com/arbadacarbaYK/gitnostr)**.

**Components:** `git-nostr-bridge` · `git-nostr-ssh` · `git-nostr-db` · `git-nostr-cli` (`gn`)

**Docs (canonical):** [README](https://github.com/arbadacarbaYK/gitnostr) · [SSH_GIT_GUIDE.md](https://github.com/arbadacarbaYK/gitnostr/blob/main/SSH_GIT_GUIDE.md) · [docs/](https://github.com/arbadacarbaYK/gitnostr/tree/main/docs)

**gittr:** issues, PRs, commits, bounties, Pages — [gittr repo](https://github.com/arbadacarbaYK/gittr) · [GIT_NOSTR_BRIDGE_SETUP.md](../../docs/GIT_NOSTR_BRIDGE_SETUP.md)

**Build here (monorepo):**

```bash
make git-nostr-bridge git-nostr-ssh   # bridge
make git-nostr-cli                    # gn
```
